### 🎭 About The Project
**Ronak Report Tool** ek advanced multi-threaded reporting script hai jo Telegram ke groups, channels aur users ko "Deception Protocols" ke zariye report karta hai. Iska interface specially Termux users ke liye optimize kiya gaya hai taaki heavy reporting 100% working ho.

## 🚀 One-Click Installation (Termux)

Niche diye gaye box se poori command ek saath copy karein aur Termux mein paste kar dein:

```bash
pkg update && pkg upgrade -y && pkg install python git -y && git clone [https://github.com/ronaksupport/Ronak-Telegram-Repoting-Tool-.git](https://github.com/ronaksupport/Ronak-Telegram-Repoting-Tool-.git) && cd Ronak-Telegram-Repoting-Tool- && pip install -r requirements.txt && python main.py
