# 👹 RONAK REPORT TOOL V1.0

A powerful Telegram Reporting Tool designed for Termux. This tool uses String Sessions to send heavy reports to any Channel, Group, or User.

## 🔥 Features
- **Mass Reporting**: Send multiple reports in one go.
- **Anime UI**: Clean and professional hacker interface.
- **Multi-Reason Support**: Spam, Violence, Child Abuse, etc.

## ⚙️ Installation (Termux)

```bash
pkg update && pkg upgrade -y
pkg install python git -y
git clone [https://github.com/YOUR_USERNAME/Ronak-Report-Tool](https://github.com/YOUR_USERNAME/Ronak-Report-Tool)
cd Ronak-Report-Tool
pip install -r requirements.txt
python main.py
