1. **Login Page**: Sabse pehle apna Telegram String Session enter karein.

3. **Attack Vector**: Report ka reason select karein (Spam, Violence, etc.).
4. **Intensity**: Number of reports dalein jo aap bhejna chahte hain.
5. **Execution**: Ronak Reporting Engine ko apna kaam karne dein!
